# EBS 볼륨 확장 하는 방법

$ lsblk

명령어를 통해 파티션 영역 확인

파티션 확장을 위하여

$ sudo growpart /dev/xvda 1

위의 명령어 입력

$ lsblk

명령을 실행하여 파티션이 확장 되었는지 확인

$ df -h

마운트 영역 확인

디스크 공간 확장을 위해

$ sudo resize2fs /dev/svda1

$ df- h

디스크 공간 확인

볼륨 확장 적용 방법 >>

  1. lsblk 를 통해 볼륨 파티션 확인

500G 중 250G만 사용되고 있음을 확인

  1. 명령어를 통해 파티션 확장

$ sudo growpart /dev/nvme0n1 1

  1. 파티션 확장 확인

  2. df -h 를 통해 마운트 영역 확인

  3. 명령어를 통해 디스크 공간 확장

$ sudo resize2fs /dev/nvme0n1p1
  1. 다시한번 df -h 를 통해 확인
